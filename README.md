# Mumbai Cafe Finder

Mumbai Cafe Finder is a React-based web application that allows users to explore and discover cafes across Mumbai through a clean and responsive interface.

## Overview

This project is built using modern frontend technologies to provide a fast and user-friendly experience. It demonstrates component-based architecture, state management, and responsive UI design.

## Features

- Search and browse cafes
- Responsive layout for different screen sizes
- Theme toggle (light/dark mode)
- Fast development and build performance using Vite

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS

## Getting Started

### Prerequisites
- Node.js installed on your system

### Installation

1. Clone the repository:
   git clone https://github.com/janvishetty/mumbai-cafe-finder.git

2. Navigate to the project directory:
   cd mumbai-cafe-finder

3. Install dependencies:
   npm install

4. Start the development server:
   npm run dev

## Project Structure

- src/ → React components and logic
- public/ → static assets
- index.html → entry point
