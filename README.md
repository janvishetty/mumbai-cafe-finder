# ☕ Mumbai Cafe Finder

A React-based web app to explore and find cafes in Mumbai.

## 🚀 Features
- 🔍 Search cafes
- 🌙 Dark / Light mode toggle
- 📱 Responsive design
- ⚡ Fast performance using Vite

## 🛠️ Tech Stack
- React
- Vite
- JavaScript
- CSS

## ▶️ Run Locally
1. Install dependencies:
   npm install

2. Start development server:
   npm run dev
